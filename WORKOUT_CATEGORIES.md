# 80/20 Triathlon Workout Categories

This document categorizes all workout code families (Bike, Run, Swim) by type, training focus, and typical training phase usage.

## Bike

| Workout Type            | Code(s)                      | Primary Focus                                                                                 |
|-------------------------|-----------------------------|-----------------------------------------------------------------------------------------------|
| Aerobic Intervals / Progression (Endurance) | CAe, CAP                   | Sub-threshold aerobic intervals and progressive endurance; build aerobic base and durability. |
| Anaerobic Intervals     | CAI, CAn                    | Short, hard intervals at/above VO₂max; improve anaerobic capacity, VO₂max, and high-end power. |
| Accelerations / Leg Speed | CA1–CA9                    | Short accelerations; leg speed, neuromuscular recruitment, smooth pacing and shifting.       |
| Cruise Intervals        | CCI                         | Long, steady intervals just below threshold; efficient tempo/endurance work.                 |
| Critical Power          | CCP                         | Sustained work near critical power/FTP; threshold strength and race-pace durability.         |
| Depletion               | CD                          | Long, low-glycogen/endurance rides; fatigue resistance and fat-oxidation.                    |
| Descending Intervals    | CDI                         | Sets that get faster or shorter recoveries; pacing control and finish-strong ability.        |
| Foundation              | CF                          | General aerobic base rides at low–moderate intensity; technique and comfort on the bike.     |
| Fast Finish             | CFA, CFF                    | Endurance/tempo ride with strong final segment; practice negative splits and late-race surges. |
| Force Intervals         | CFo                         | Low-cadence, high-torque efforts; muscular strength and climbing power.                      |
| Mixed Intervals         | CIM, CMI                    | Varied intervals (durations and intensities); all-round conditioning and race-like variability. |
| Sprint Intervals        | CIR                         | Very short, maximal sprints with full recovery; neuromuscular power and top-end speed.       |
| Progression Intervals   | CPI                         | Interval sets that progress in pace or duration; pacing discipline and fatigue resistance.   |
| Power Repetitions       | CPR                         | Repeated high-power surges; anaerobic power and repeatability under fatigue.                 |
| Recovery                | CRe                         | Very easy spins; promote circulation and recovery without adding stress.                     |
| Speed Play (Fartlek)    | CSP                         | Free-form surges in an otherwise easy ride; reactivity and fun speed work.                   |
| Speed Repetitions       | CSR                         | Short fast reps at high cadence; speed, form, and neuromuscular sharpness.                   |
| Steady State            | CSS                         | Long, steady efforts around upper aerobic / low tempo; muscular endurance.                   |
| Tempo                   | CT                          | Classic tempo riding below threshold; sustainable “comfortably hard” efforts.                |
| Threshold               | CTR                         | Work near/at lactate threshold; raise FTP and threshold HR.                                  |
| Time Trial              | CTT                         | Continuous or broken race-pace efforts; pacing and mental toughness.                         |
| Variable Intensity      | CVI                         | Alternating intensities within sets; handling surges, race-like stochastic efforts.          |
| VO₂ Max                 | CVO2M                       | Short high-intensity Z5 efforts; improve maximal aerobic capacity and oxygen uptake.         |
| Endurance               | EC                          | Steady low–mid Z2 endurance; core base mileage.                                              |
| Easy                    | EZC                         | Very easy spins; active recovery or aerobic maintenance.                                     |
| Lactate Intervals       | LIC                         | High-intensity intervals near/above threshold; lactate tolerance and buffering.              |
| Over Under              | OUC                         | Alternating slightly above and below threshold; lactate clearance and threshold robustness.  |

## Run

| Workout Type                 | Code(s)                     | Primary Focus                                                                                      |
|------------------------------|-----------------------------|----------------------------------------------------------------------------------------------------|
| Aerobic Intervals           | RAe                         | Repeated aerobic intervals below threshold; build aerobic base and fatigue resistance.            |
| Anaerobic Intervals         | RAI, RAn                    | Hard intervals at/above VO₂max; speed and anaerobic capacity.                                     |
| Accelerations / Strides     | RA                          | Short fast pickups during easy runs; leg speed, form, and neuromuscular sharpness.               |
| Cruise Intervals            | RCI                         | Threshold-adjacent intervals; sustain strong but controlled pace.                                 |
| Critical Velocity            | RCV                         | Work near critical velocity; improve sustainable fast pace and aerobic power.                     |
| Depletion Runs              | RD                          | Long, lower-glycogen endurance; fatigue resistance and fuel-usage efficiency.                    |
| Descending Intervals        | RDI                         | Intervals that get faster or with shorter recoveries; pacing control and finishing strength.     |
| Foundation Runs             | RF                          | General easy–moderate runs; basic aerobic conditioning and durability.                            |
| Fast Finish Runs            | RFF                         | Easy–moderate run with last portion at faster pace; negative split ability.                      |
| Fartlek                     | RFR                         | Unstructured surges within an easy run; speed, fun, and responsiveness.                          |
| Half-Marathon Pace          | RHM                         | Work around half-marathon pace; race-specific prep for that distance.                            |
| Heart Rate Structured       | RHR                         | Runs guided by HR zones; discipline in staying in prescribed zones.                              |
| Long Runs                   | RL                          | Long continuous endurance; key long-distance aerobic stimulus.                                   |
| Long Speed Play             | RLS                         | Long run with fartlek-style surges; mix of endurance and speed.                                  |
| Mixed Intervals             | RMI                         | Varied intervals (pace and duration); all-around conditioning, race-like fluctuations.           |
| Marathon Pace               | RMP                         | Sustained or broken marathon-pace efforts; specific prep for marathon and long-course tri.      |
| Progression Run             | RP                          | Run that steadily increases from easy to moderate/hard; pacing and strong finishing.             |
| Progression Fartlek         | RPF                         | Fartlek run that also trends faster over time; both speed and endurance under changing pace.     |
| Progression Intervals       | RPI                         | Interval sets that increase in pace or duration; fatigue resistance and psychological toughness. |
| Recovery Runs               | RRe                         | Very easy, short runs; promote recovery with minimal stress.                                     |
| Short Intervals             | RSI                         | Short, fast reps with moderate recovery; speed and neuromuscular development.                    |
| Speed Play                  | RSP                         | Paces and intervals vary; similar to fartlek but often more structured.                          |
| Steady State                | RSS                         | Sustained, moderate-hard pace below threshold; strong aerobic stimulus.                          |
| Tempo                       | RT                          | Classic tempo around lactate threshold; “comfortably hard” sustained work.                       |
| Time Trial                  | RTT                         | Maximal or near-maximal sustained efforts; gauge fitness and practice race-pace execution.       |
| Variable Intensity          | RVI                         | Alternating paces within sets or runs; race-specific variability and resilience.                 |
| VO₂ Max                     | RVO2M                       | High-intensity Z5 work; maximize aerobic power and running speed.                                |
| Cross-Training              | RXT                         | Non-run activities (bike, swim, etc.); maintain aerobic fitness with reduced run stress.         |
| 5K-Specific                 | R5K                         | Work tuned to 5K race pace; high intensity, shorter distance focus.                              |
| 10K-Specific                | R10K                        | Work tuned to 10K race pace; threshold/VO₂ blend.                                               |
| Easy Runs                   | ER                          | Comfortable aerobic runs; day-to-day mileage and recovery.                                      |
| Easy + Fast Finish          | ERFF                        | Mostly easy with short faster finish; economical speed while preserving low stress.             |
| Long Finish Runs            | LFR                         | Long runs finishing at faster pace; simulate late-race fatigue and closing hard.                |
| Long Intervals              | LIR                         | Longer repeats (e.g. km or mile reps); threshold/endurance blend.                               |
| Outdoor (terrain/conditions) | OUR                        | Runs emphasizing outdoor/variable terrain; handling hills, wind, and real-world surfaces.       |
| Warmup Runs                 | WR                          | Short, easy warmup sessions; prepare for harder workouts or races.                              |

## Swim

| Workout Type                 | Code(s)                     | Primary Focus                                                                                      |
|------------------------------|-----------------------------|----------------------------------------------------------------------------------------------------|
| Aerobic Intervals           | SAe                         | Repeated aerobic swim sets; build general swim endurance.                                         |
| Broken Swims                | SBB                         | Broken distances (e.g. 800 as 8×100) with short rest; race-pace endurance and pacing.            |
| Cruise Intervals            | SCI                         | Steady, sub-threshold cruise sets; strong but controlled aerobic/tempo work.                      |
| Critical Pace               | SCP                         | Work near critical swim pace; sustainable fast pace and threshold.                                |
| Endurance                   | SE                          | Longer continuous or large-block sets at moderate intensity; base swim endurance.                |
| Easy Endurance              | SEE                         | Low-intensity longer swims; easy endurance and technical focus under low stress.                 |
| Endurance Recovery          | SER                         | Very easy endurance-oriented swims; active recovery with emphasis on relaxed form.               |
| Foundation                  | SF                          | Fundamental technique + basic aerobic; drills combined with easy swimming.                        |
| Short Intervals             | SIS                         | Shorter repeats (25–100m); speed and neuromuscular sharpness.                                    |
| Lactate Intervals           | SLI                         | High-intensity sets above threshold; lactate tolerance and race-pace toughness.                  |
| Mixed Intervals             | SMI                         | Varied sets (pace, distance, sometimes strokes); overall conditioning and adaptability.          |
| Recovery                    | SRe                         | Very easy, short sessions; promote recovery and maintain feel for the water.                     |
| Short Sprint Intervals      | SSI                         | Very fast 25–50m reps with full recovery; top-end speed and stroke power.                        |
| Speed Play                  | SSP                         | Fartlek-style speed changes within a session; fun variability and race-like changes of pace.     |
| Tempo                       | ST                          | Sustained or broken sets at strong but sub-max effort; threshold-adjacent swim fitness.          |
| Threshold Intervals         | STI                         | Repeats at threshold pace with limited recovery; raise swim threshold and tolerance to effort.   |
| Time Trial                  | STT                         | All-out or near-all-out efforts over a set distance; assess fitness and practice race pacing.    |
